# Change Log

All notable changes to this extension will be documented in this file.

## [0.0.1] - 2025-01-XX

### Added
- Initial release
- Generate alt text for images using AI
- Support for HTML img tags and React Image components
- Support for local files, relative paths, and URLs
- Automatic image resizing to prevent payload errors
- Multiple input methods (command palette, editor selection, file explorer)
- Multi-language support
- API key configuration
- Configurable image size limits and quality settings
